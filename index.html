RogueCanvas

Why the name? It is a RogueLike inspired game built rending all the fun action using HTML Canvas. The rest of this game is built using plain JavaScript. Each map is randomly generated using concepts from Conways Game of Life. After which I use a function to 'flood' the explorable cave and if it is not big enough (atleast 800 tiles to step on) then it will rerender a new map. I hope you enjoy it as much as I enjoy making it.

Thanks for playing! Now go kill me some creatures!
